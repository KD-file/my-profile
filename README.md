# My Personal Profile

A full-stack personal portfolio website with user authentication, an admin dashboard, and a contact form that stores messages in MongoDB Atlas.

🔗 **Live Demo:** https://my-profile-14j3.onrender.com
🔗 **git:** https://github.com/KD-file/my-profile.git

---

## ✨ Features

- **Home Page** – Clean landing page with a welcome greeting and call-to-action buttons.
- **About Page** – Personal bio, skills, education, and hobbies.
- **Contact Page** – Functional contact form that saves messages to MongoDB. Includes social media links (Facebook, YouTube).
- **Register Page** – User registration with password hashing (bcrypt). First user to register becomes the **admin**.
- **Login Page** – Secure login with JWT (JSON Web Token) authentication.
- **Admin Dashboard** – Exclusive to the admin account. Displays all messages sent via the contact form.
- **Cloud Database** – All user and message data is stored in MongoDB Atlas.
